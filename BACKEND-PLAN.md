# План бекенда: Голосовой ассистент

## Данные для хранения
- Пользователи: ID, настройки, история команд.
- Команды: Текст, ответы, timestamp.
- Интеграции: Ключи API, настройки сервисов.

## API
- Speech Recognition API (Google/Amazon).
- Text-to-Speech API.
- Database API для CRUD операций.

## Кто что видит и редактирует
- Пользователи: свои данные, команды.
- Админы: все данные, настройки.

Только необходимое для приложения.
