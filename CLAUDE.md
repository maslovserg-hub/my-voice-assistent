# Правила работы с проектом

## Общие правила
- Все страницы делаем в одном HTML-файле, без внешних библиотек.
- Дизайн: тёмный фон, светлый текст, акцентный цвет — оранжевый.
- Адаптивная вёрстка — всё должно хорошо выглядеть на мобильных устройствах.
- Стиль текста: простой, дружеский, без канцелярита.
- Перед созданием нового файла — сначала расскажи, что планируешь сделать.

## Дополнительные рекомендации
- Используй чистый HTML, CSS и JavaScript.
- Обеспечивай плавные анимации и интерактивные элементы.
- Проверяй адаптивность на различных устройствах.
- Поддерживай голосовой ввод для заданий, где возможно.
