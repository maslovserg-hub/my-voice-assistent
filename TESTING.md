# TESTING.md — тестирование VoiceType

## Запуск для разработки

```bash
voicetype.bat
```

## Тест микрофона / модели отдельно

```bash
test_mic.bat
```

## Чеклист: базовая функциональность

- [x] Right Ctrl активирует запись, оверлей появляется
- [x] Полоски анимируются по уровню звука
- [x] Текст вставляется порциями после паузы ~1.5с
- [x] Повторное Right Ctrl останавливает, вставляет остаток
- [x] Esc отменяет запись без вставки текста
- [x] Оверлей скрывается после остановки
- [x] Второй запуск показывает предупреждение и закрывается

## Чеклист: качество распознавания

- [x] Русская речь с пунктуацией: «Однажды в студёную зимнюю пору...»
- [x] Скороговорки: «Четыре чёрненьких чумазеньких чертёнка...»
- [x] Английские слова в русской фразе
- [x] Латинские артефакты без гласных удаляются
- [x] Артефакт «ы» в начале чанка удаляется
- [ ] Тест на планшете Samsung Galaxy Tab Pro S

## Чеклист: совместимость приложений

- [ ] Браузер (Chrome/Edge) — адресная строка, поля форм
- [ ] Telegram Desktop
- [ ] Блокнот / Notepad
- [ ] Word / LibreOffice
- [ ] VS Code

## Чеклист: трей и автозагрузка

- [x] Трей-иконка появляется при запуске
- [x] "Запускать с Windows" — добавляет/убирает из реестра
- [x] "Закрыть" — корректно завершает приложение

## Чеклист: мастер первого запуска

- [x] При отсутствии модели показывается wizard
- [x] "Выбрать папку" — принимает папку с готовой моделью
- [x] "Скачать модель" — скачивает с прогрессом, повторяет при обрыве
- [x] Пустые/неполные файлы от прошлых попыток удаляются автоматически

## Чеклист перед релизом

- [x] .exe запускается без установленного Python
- [x] Нет мелькающих консольных окон при распознавании
- [ ] Тест на чистой машине без C:/gigaam_cache
- [ ] Тест на Samsung Galaxy Tab Pro S (4 ГБ RAM, Core m3)

## Известные ограничения

| Ограничение | Значение |
|-------------|---------|
| RAM в работе | ~1.5 ГБ |
| Размер .exe | ~170 МБ |
| Размер модели | ~650 МБ |
| Задержка вставки | ~1-2 сек после паузы |
| Платформа | Windows 10+ только |
