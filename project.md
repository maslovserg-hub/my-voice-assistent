# Описание проекта

## Что это за проект
Это проект голосового ассистента, предназначенный для помощи пользователям в повседневных задачах через голосовые команды. Система обрабатывает речевые запросы, выполняет действия и предоставляет обратную связь. Проект включает веб-интерфейс для взаимодействия и интеграцию с различными сервисами.

## Для кого он
Проект предназначен для пользователей, которые предпочитают голосовое взаимодействие с технологиями, включая разработчиков, бизнес-пользователей и обычных потребителей, нуждающихся в удобном доступе к информации и управлению задачами.

## Какие страницы уже есть
На данный момент в проекте нет готовых страниц. Есть только файл агента (.agent.md) для управления документацией.

## Какие страницы планируются
- Визитка (готово, но не реализовано)
- Лендинг (готово, но не реализовано)
- Страница услуг
- Страница отзывов
- Страница тарифов
- Страница контактов
- Форма обратной связи
- Портфолио с примерами работ

## Какой стиль дизайна используется
Дизайн использует тёмный фон, светлый текст и акцентный оранжевый цвет. Вёрстка адаптивная, стиль текста простой и дружеский, без канцелярита. Все страницы делаются в одном HTML-файле без внешних библиотек.
