# Голосовой ввод в любую программу

Программа позволяет вводить текст голосом в любое активное поле на компьютере с помощью горячей клавиши.

## Запуск

1. Установите зависимости: `pip install SpeechRecognition pynput`
2. Запустите скрипт: `python voice_input.py`
3. Скрипт работает в фоне.

## Использование

- Запустите `run_voice_input.bat` двойным щелчком.
- Откроется окно командной строки, в котором будет сообщение:
  - `Запущен голосовой ввод. Нажмите правый Ctrl или F9 для активации`
- Перейдите в любое текстовое поле в другой программе.
- Нажмите **правый Ctrl** для начала записи.
- После звукового сигнала говорите текст.
- Текст будет автоматически печататься в активное поле по мере распознавания.
- Нажмите **правый Ctrl** ещё раз, чтобы остановить запись.

Если клавиша не работает, закройте окно и запустите `run_voice_input.bat` правой кнопкой мыши с помощью "Запуск от имени администратора".

Если появится ошибка `Could not find PyAudio`, выполните в командной строке:

```
python -m pip install pyaudio
```

## Возможные ошибки и решения

- **Ошибка вставки текста**: Если текст не вставляется в поле, попробуйте вставить его вручную. Некоторые программы могут блокировать автоматический ввод.
- **Ошибка распознавания**: Проверьте интернет-соединение, так как используется Google API. Если проблема с микрофоном, проверьте список микрофонов.
- **Горячая клавиша не работает**: Запустите скрипт от имени администратора. Убедитесь, что не конфликтует с другими программами.
- **Не удалось распознать речь**: Говорите громче, проверьте микрофон. Попробуйте короче фразы.

## Как улучшить качество распознавания

- Говорите чётко и не торопясь.
- Находитесь в тихом помещении.
- Не говорите во время настройки шума после сигнала.
- Если используется не тот микрофон, запустите скрипт с параметром:

```
python voice_input.py --list-mics
```

В этом случае будет показан список устройств. После этого можно выбрать нужный микрофон по индексу:

```
python voice_input.py --mic-id 2
```

или:

```
python voice_input.py -m 2
```

## Требования

- Микрофон
- Интернет (для Google Speech Recognition)
- Python 3.x

Закройте терминал или нажмите Ctrl+C в терминале скрипта.
